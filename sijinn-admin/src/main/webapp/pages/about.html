Simple Java Implementation of Neural Network

  • Network structure: perseptron

  • Training strategies:

    • Gradient Descent (Batch, Stochastic)

    • Genetic Breeding

  • Training algorithms:

    • BPROP - Back PROPagation

    • QPROP - Quick backPROPagation

    • RPROP - Resilient backPROPagation

    • GENE - GEnetic NEural

  • features: create new network, edit existent, set training data/strategy/algorithm and run training in 'shadow' mode

  • status: 1.1.0.alfa - work in progress

Provided

  • BackEnd: ClassHidra (MVC, RESTful, Scheduler)

  • FrontEnd: AngularJS 1.5, Bootstrap, BootstrapNativeJS

  • Neural Networks: SIJINN
