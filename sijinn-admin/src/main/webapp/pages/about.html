Simple Java Implementation of Neural Network

  • network structure: perseptron

  • training algorithms: BPROP, QPROP, RPROP

  • this tool allows to create new networks, edit existent, prepare training scenarios and run training in scheduled mode

  • all your structures can be saved - file system is used as storage

  • status: 1.0.1.alfa - work in progress

Provided

  • BackEnd: ClassHidra (RESTful)

  • Neural: SIJINN

  • FrontEnd: AngularJS 1.5
